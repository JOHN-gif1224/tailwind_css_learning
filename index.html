BUSY BAR est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distracttion sur votre téléphone et PC. Entièrement personnalisable, open-source et prêt pour la maison connectée.

Outils de productivité

Blocage des distraction, message BUSY personnalisé, minuteur de concentration, synchronisation multiplateforme

Applications et intégrations

Bibliothèque d'applications, connexions à des logiciels tiers, intégrations avec les événements de calendrier et les appels

Support maison connectées

Contrôle des appareils connectés, automatisation des tâches, intégration avec les assistants vocaux

Pour les développeurs

API ouverte, documentation complète, exemples de code
